# Train-Info
``` Gives Information about Indian Railway Trains. ```

### Currently Active Features.
1. Get PNR Status
2. Get Train Name by Number or Vice-Versa.
3. Running Days of a Train.

### Under Development.
1. Live Train Status.
2. Seat Availability.
3. Train Fare Enquiry.
